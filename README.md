# Generador de Contraseñas Seguras

**Generador de Contraseñas Seguras** es una herramienta web que permite crear contraseñas aleatorias y fuertes. El generador ofrece opciones de personalización para ajustar la longitud y los caracteres incluidos en la contraseña, como mayúsculas, números y símbolos especiales.

Esta herramienta está diseñada para ayudar a generar contraseñas robustas y seguras para proteger tus cuentas en línea.

---

## Características

- **Generación de contraseñas aleatorias** con configuraciones personalizables.
- **Longitud de la contraseña** ajustable entre 8 y 32 caracteres.
- Opción para incluir:
  - **Mayúsculas**
  - **Números**
  - **Símbolos especiales** (e.g., `!@#$%^&*()`)
- **Copiar la contraseña** al portapapeles con un solo clic.

---

## Tecnologías Usadas

- **HTML**: Para la estructura básica de la página.
- **CSS**: Para el diseño visual y la estética de la interfaz.
- **JavaScript**: Para la lógica de generación de contraseñas y la interacción con el usuario.
